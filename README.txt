This application allows one to keep track of a swiss style tournament.

Please review the file tournament.py to understand the functions that are being used

The appropriate way to use this file is to test the functions in tournament.py
by using running the file tournament_test.py

tournament_test.py tests the database functions of tournament.py mock tournament
to ensure that they function properly in a tournament situations.

Directions:
  1.  Create a database using the tournament.sql file
      - instructions are commented out in the file
  2.  To test that everything is working properly, run tournament_test.py
  3.  You are now ready to use the functions listed in tournament.py on your own tournament
